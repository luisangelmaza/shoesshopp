## Reemplazar con Imágenes Propias

Si deseas usar tus propias imágenes:

1. Coloca las imágenes en esta carpeta (`app/images/`)
2. Actualiza las URLs en `app/css/styles.css` buscando las variables `--bg-*`
3. Asegúrate de que las imágenes tengan buena resolución (mínimo 1920px de ancho para fondos)

## Formatos Recomendados

- **Fondos grandes**: JPG o WebP (optimizado)
- **Patrones**: PNG con transparencia o SVG
- **Iconos**: SVG o PNG
